🤝 ブランドアイデンティティ
「バディ」の定義
相棒的役割。ダイビング仲間、信頼できるインストラクター、なんでも相談できる相手。 タメ口ではなく自然なやりとり。
サービスの本質
コンシェルジュサービス
「探しに行く」→「紹介してもらう」へのパラダイムシフト
差別化ポイント
「自分の意見や要望を伝えられるサービス」
文字ベースの要望 → AIマッチング → 最適提案
🎯 ターゲット戦略
メインターゲット
20代後半〜30代前半女性
沖縄ダイビング初心者・経験浅い層
事業戦略
B案：幅広いユーザー獲得
ショップ広告収入モデルのため、アクティブユーザー数が重要
成功指標
短期：満足度・使いやすさ
長期：リピート利用率
💕 感情的価値・体験設計
核心的価値
「沖縄の海で輝く私」
沖縄ブランドの海で潜る自分への憧れ
感情変化の設計
不安 → ワクワク → もっと知りたい
楽しみに変わる瞬間：全予約完了時、行きたい離島発見時
ユーザーニーズ
普段と違う経験をしたい、透明度の高い海、沖縄ブランドの海で潜る自分に酔いたい
🛤️ 動線・機能設計
Web vs LINE の役割
Web：情報閲覧（受動的）
LINE：マッチング・相談（能動的）
理想の動線
サイト訪問 → サービス本質説明 → 信頼獲得 → LINE登録 → アンケート → 本格マッチング
LINE登録のタイミング
サービス概要を見て、情報を見て信頼できると感じたら「相談したい」と思うタイミング
